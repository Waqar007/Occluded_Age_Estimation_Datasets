# Occluded_Age_Estimation_Datasets

This repository contains the occluded versions of three age estimation datasets.

## Datasets
- FG-NET
- APPA-REAL
- MORPH

## Occlusion Types
- Sunglasses
- Surgical Masks
- Mouth Scarves
- Head Coverings
- Forehead Coverings
- Tattoos
- Beard-Moustache
- Periocular Makeup
