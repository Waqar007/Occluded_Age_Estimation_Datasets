# Occluded_Age_Estimation_Datasets

This repository contains the occluded versions of three age estimation datasets. Eight type of occlusions are provided including Sunglasses, Surgical Masks, Mouth Scarves, Head Coverings, Forehead Coverings, Tattoos, Beard-Moustache, and Periocular Makeup

## Datasets
- FG-NET
- APPA-REAL
- MORPH
