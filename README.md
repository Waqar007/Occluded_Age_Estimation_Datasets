# Occluded_Age_Estimation_Datasets

This repository contains the occluded versions of three age estimation datasets.

## Datasets
- FG-NET-O8
- APPA-REAL-O8
- MORPH-O8

## Occlusion Types
- Sunglasses
- Surgical Masks
- Mouth Scarves
- Head Coverings
- Forehead Coverings
- Tattoos
- Beard-Moustache
- Periocular Makeup
